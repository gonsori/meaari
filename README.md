# meaari prototype

메아리 1차 MVP의 첫 진입 온보딩과 시작 화면 프로토타입입니다.

## 포함 범위

- 서비스명: `meaari / 메아리`
- 핵심 문구: `저장한 문장이 필요한 순간 다시 돌아와요`
- 데모용 상기 채널 선택: 위젯, 잠금화면, 앱푸시
- CTA: `문장 저장해보기`
- 모바일 우선 UI
- Vercel 배포 가능한 Vite React 구조

## 제외 범위

- 실제 OS 위젯 설정
- 실제 잠금화면 위젯 구현
- 실제 앱푸시 권한 요청
- 문장 입력/저장 플로우
- GA4 이벤트 연동

## 실행

```bash
npm install
npm run dev
```
